Where to find game music?

You need the assets folder from your Minecraft installation.

  • Windows: %appdata%\.minecraft\assets
  • Mac: ~/Library/Application Support/minecraft/assets
  • Linux: ~/.minecraft/assets

Select the root assets folder (the one containing "indexes" and "objects").
