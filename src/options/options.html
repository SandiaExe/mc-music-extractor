Guide: Where to find game music (by operating system)

Minecraft Java Edition — You need the assets folder from your installation. When choosing a folder, you need the sound files (usually the assets folder).

  • Windows: %appdata%\.minecraft\assets (type %appdata% in the File Explorer and go to .minecraft\assets).
  • Mac: ~/Library/Application Support/minecraft/assets (Go menu → Go to Folder and paste the path).
  • Linux: ~/.minecraft/assets (show hidden folders in your file manager and go to .minecraft\assets).

Steps: 1) Open the path for your OS. 2) Click "Select Assets Folder" and choose the assets folder (the one containing indexes and objects). 3) Wait for the import to finish and restart the extension if prompted.
